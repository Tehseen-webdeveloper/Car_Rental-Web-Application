# 🚗 Car Rental Web Application

A modern, scalable, and high-performance Car Rental Management System built with the MERN stack and Vite. This application allows users to browse, search, and book luxury and economy vehicles seamlessly.

---

## 🏗 Architecture & Tech Stack

This project follows **Clean Architecture** principles to ensure scalability and maintainability.

- **Frontend:** React.js, Vite, Tailwind CSS (for responsive UI/UX)
- **State Management:** Context API / Redux Toolkit
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT (JSON Web Tokens) with HttpOnly Cookies
- **Deployment:** Vercel (Frontend) / AWS EC2 (Backend)

---

## 🚀 Key Features

- **User Authentication:** Secure Login/Signup with role-based access control (Admin & User).
- **Dynamic Fleet Management:** Admin can add, update, or remove cars from the inventory.
- **Advanced Search & Filter:** Filter by car type, price range, and availability.
- **Booking System:** Real-time booking with date validation.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop views.
- **Performance:** Fast TBT (Total Blocking Time) and high Lighthouse scores using Vite.
